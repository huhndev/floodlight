<p align="center">
  <code>floodlight</code> is a minimal, fast, no-JavaScript <a href="https://gohugo.io">Hugo</a> theme with a single neon accent.
</p>

---

### Usage

```
git submodule add https://github.com/huhndev/floodlight.git themes/floodlight
```

Set `theme = "floodlight"` in your `hugo.toml`. Demo: [www.huhn.dev](https://www.huhn.dev).
